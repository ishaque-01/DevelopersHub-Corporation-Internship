# DevelopersHub-Corporation-Internship

A centralized repository for tracking Data Science and Analytics tasks during my internship at DevelopersHub Corporation.

## Internship Progress

### Task 1: Basic Data Exploration and Visualization
* **Objective:** Load a standard dataset and perform initial exploration and visualization.
* **Actions:**
    * Set up a Miniconda environment for Data Science.
    * Loaded the iris dataset using the seaborn library.
    * Performed data inspection using .shape, .columns, and .head().
    * Created visualizations: Scatter Plot, Histogram, and Box Plot to identify outliers and distributions.

### Task 2: Credit Risk Prediction
* **Objective:** Predict the likelihood of a loan applicant defaulting using classification models.
* **Actions:**
    * Preprocessed data and handled categorical features using Label Encoding.
    * Split the dataset into 80% training and 20% testing sets.
    * Implemented a Logistic Regression model for binary classification.
    * Evaluated performance using an Accuracy Score, Confusion Matrix, and Classification Report.
* **Tools Used:** Scikit-learn, Pandas, NumPy.

### Task 3: Customer Churn Prediction
* **Objective:** Predict bank customer churn based on demographic and financial factors.
* **Actions:**
    * Cleaned data by removing non-predictive columns like RowNumber and CustomerId.
    * Scaled numerical features using StandardScaler for improved model accuracy.
    * Trained a Random Forest Classifier to identify patterns in customer behavior.
    * Measured success using Accuracy Score and ROC AUC Score.
* **Tools Used:** Scikit-learn, Matplotlib, Seaborn.
